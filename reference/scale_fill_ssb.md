# Create discete SSB colours

Create discete SSB colours

## Usage

``` r
scale_fill_ssb(name = "main")
```

## Arguments

- name:

  Name of the palette to use. Default is 'main'. Choose between:  
  'main', greens', 'blues', 'primary', 'secondary', 'shade1',
  'shade2','shade3', 'shade4', 'shade5'

## Value

ggproto object for using in ggplot to change discrete colours
